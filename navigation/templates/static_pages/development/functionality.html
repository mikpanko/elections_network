Нужна для определения уровня доверия между юзерами и площадки к юзерам. По умолчанию пользователи будут не верифицированными. Изначально будет четыре типа верификации, с возможностью добавления новых по мере необходимости:
    • через краудсорсинг по пользователям (сколько пользователей площадки подтвердили, что знают лично)
    • через партнера - организацию на площадке
    • через ссылку на внешний сайт/блог/аккаунт в соцсети, владение которым можно подтвердить через доступ к ссылке, отправленной владельцу сайта/блога/аккаунта в соцсети (пройдя по которой, надо будет залогиниться на сайте, чтобы завершить верификацию)
    • через высылку фото или сканов документов, подтверждающих статус
